# Box Game Download Page

A professional, sleek, and modern web page for downloading the Box Game.

## Game Controls

- "A" for left movement
- "D" for right movement
- "W" for stopping

## How to Deploy to Vercel

1. Install Vercel CLI:
   ```
   npm install -g vercel
   ```

2. Login to Vercel:
   ```
   vercel login
   ```

3. Deploy the project:
   ```
   vercel
   ```

4. For production deployment:
   ```
   vercel --prod
   ```
